# Project: Simple To-do List

## Brief
A REST API + minimal frontend to manage personal tasks. Users can create, read, update, delete tasks, mark as complete, filter by priority, and search by title.

## Scope (3-5 features)
1. Task CRUD (Create, Read, Update, Delete)
2. Mark task as complete/incomplete
3. Set priority (Low, Medium, High)
4. Filter tasks by priority
5. Search tasks by title (full-text)

## Out of scope
- User authentication
- Due dates (nice-to-have but skip for time)
- Tags/labels

## Target users
Individual user (single-user app)
